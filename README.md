## 🖥️ 프로젝트 소개
팀 프로젝트로 진행한 테니스 코트 대여 & 운동 용품경매 사이트입니다.

## 🕰️ 개발 기간
* 2023.06.12 ~ 2023.07.21

## ⚙️ 개발 환경
- `Java 1.6`, `HTML/CSS`, `Javascript`
- **IDE** : Spring Framwork 3.1.1, Sql Developer
- **Database** : Oracle DB(11g)
- **Server** : Apache Tomcat 9.0
- **ORM** : Mybatis

<p align="center" width="100%">
